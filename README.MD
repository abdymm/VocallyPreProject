## Getting Started

### Quick Start
1. Clone the repo.
2. Install package ```$ yarn install```
3. Run iOS app with ```$ yarn ios```
4. Run Android app with ```$ yarn android```

### Stacks
- Typescript
- Navigation: React Navigation
- React-native-maps
- State Management: Context API
